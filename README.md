# Retail Sales Data Agent

## Project Objective
This project uses Databricks SQL and a data agent to analyse a retail sales dataset.

## Tools Used
- Databricks
- SQL Editor
- GitHub

## Dataset Overview
Retail sales transaction dataset containing customer details, product categories, sales amounts, and transaction dates.

## Steps Followed
- Uploaded dataset to Databricks
- Created retail_sales_data table
- Ran SQL queries
- Configured agent instructions
- Validated results

## Agent Instructions
The agent was instructed to answer retail sales questions using the uploaded dataset.

## Sample Questions Tested
- Average customer age
- Customer count by gender
- Highest transaction dates

## Key Insights
The dataset revealed customer demographics and sales transaction patterns.

## Conclusion
The project demonstrated data analysis using Databricks SQL and a retail data agent.
